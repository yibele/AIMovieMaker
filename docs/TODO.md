# 项目 TODO 清单

> 结合现有需求，将同类事项合并为 4 个主线，方便排期和推进。

## 1. Flow 节点 & 工具栏体验
- 视频节点多功能 toolbar：下载、重新生成、删除、高清放大（Flow API 待确认）、尾帧截取。
- 图片节点 toolbar 升级：再次生成、类似图片推荐、固定放大/缩小按钮、适配/最小尺寸预设。
- TextNode 提示词联动：引入提示词库（以 TextNode 节点承载），支持单/多节点输入时的自动连接与同步更新。

## 2. 工作流 & 项目管理
- 项目管理面板：欢迎页创建/切换项目，获取并同步 Flow `projectId`。
- 工作流保存与分享：基于 IndexedDB 的本地草稿 + 导出 ZIP 方案，支持分享导入。
- Flow 同步能力：删除操作与 Flow 保持一致；保留生成历史记录，便于回溯。

## 3. 资源与风格库体系
- Flow 图片库接入：浏览与插入 Flow 侧已有素材。
- 创意资产库（姿势/分镜/人物/场景）：统一为一套可检索的资产库，支持 AI 短剧和日常创作复用。
- 风格设定资产：图片风格前缀模板 + 风格调性图库，支持一键套用与参考图生图。

## 4. AI 创作与外部助手
- AI 短剧链路：分镜生成 → 生图 → 视频生成的全链路打通。
- 外部模型接入：Gemini 助手（对话/提示词）、Sora2（视频生成）接入计划与资源评估。
