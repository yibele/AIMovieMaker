# 图标文件说明

## 需要的图标

Chrome 扩展需要以下三个尺寸的图标：

- `icon16.png` - 16x16 像素（工具栏小图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome 应用商店）

## 简单方法 1：使用 Emoji

你可以使用在线工具将 emoji 转换为图标：

1. 访问：https://emoji2png.com/
2. 输入 emoji：🍪（饼干）
3. 选择尺寸：16, 48, 128
4. 下载并重命名为 `icon16.png`, `icon48.png`, `icon128.png`
5. 放入 `icons/` 文件夹

## 简单方法 2：使用 Canva

1. 访问：https://www.canva.com/
2. 创建自定义尺寸设计（16x16, 48x48, 128x128）
3. 添加饼干 🍪 图标和紫色渐变背景
4. 导出为 PNG
5. 放入 `icons/` 文件夹

## 简单方法 3：使用图标库

1. 访问：https://icons8.com/ 或 https://www.flaticon.com/
2. 搜索 "cookie" 或"饼干"
3. 下载 PNG 格式（16x16, 48x48, 128x128）
4. 重命名并放入 `icons/` 文件夹

## 简单方法 4：不使用图标

如果暂时不想添加图标，扩展仍然可以正常工作！

1. 删除 `manifest.json` 中的 `icons` 和 `action.default_icon` 部分
2. Chrome 会使用默认的灰色图标

## 推荐设计

图标设计建议：
- 主题：饼干 🍪
- 颜色：紫色渐变（#667eea → #764ba2）
- 风格：扁平化、现代
- 背景：白色或透明

## 文件结构

```
icons/
├── icon16.png   (16x16 像素)
├── icon48.png   (48x48 像素)
└── icon128.png  (128x128 像素)
```

## 测试图标

1. 添加图标文件后
2. 重新加载扩展（chrome://extensions/）
3. 检查工具栏图标是否显示正确

---

**注意**：如果你不添加图标，扩展功能不会受影响，只是显示默认图标。
